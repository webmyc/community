## Browser AI in 60 seconds

Respira includes WebMCP directly in the plugin. You do not need a separate bridge plugin.

This page covers the **built-in Browser AI/WebMCP** surface. If you are looking for the Elementor-specific browser integration, see [Elementor Angie](/elementor-angie).

### Requirements

- WordPress admin access
- Respira for WordPress plugin
- Chrome 146+ browser
- Claude or Gemini account (or another browser AI client that supports WebMCP)

## Step 1: Install Respira

1. In WordPress Admin, go to **Plugins → Add New**
2. Search for **Respira for WordPress**
3. Click **Install**, then **Activate**

## Step 2: Open your browser AI client

Open your browser AI workflow in Chrome 146+ and connect it to your WordPress site through Respira.

<Callout kind="tip">
WebMCP is built-in, so this setup is typically faster than desktop MCP setup.
</Callout>

## Step 3: Verify the connection

Try one quick request:

- "List pages on my WordPress site"
- "Analyze SEO for my homepage"
- "Create a duplicate of page 123"

If you get results, Browser AI is ready.

## What you can do

Browser AI has access to a comparable toolset to desktop mode:

- 100+ tools total (with add-ons)
- Browser inventory follows the active plugin release contract
- Desktop MCP currently exposes `82` WordPress tools + `21` WooCommerce tools
- Same duplicate-before-edit safety workflow

## Browser AI vs Angie vs the official adapter

- **Browser AI/WebMCP**: built into Respira for Chrome browser AI clients
- **Elementor Angie**: curated browser-auth allowlist inside logged-in Elementor workflows
- **Official WordPress MCP Adapter**: WordPress-native MCP transport that can expose Respira abilities via the default/public server and the dedicated Respira server

See also:

- [Elementor Angie](/elementor-angie)
- [WordPress AI Stack Compatibility](/wordpress-ai-stack)

## Browser AI vs Desktop AI

- Browser AI: fastest setup, ideal for quick edits and team workflows
- Desktop AI: ideal for advanced automation and scriptable workflows

You can use both modes at the same time.
