Welcome. This is the public community hub for Respira for WordPress.

This repo contains docs and community resources.
It does not contain the commercial plugin source code.

## What Respira is

- A duplicate-first workflow for safe edits in WordPress
- A copilot for common site operations and builder workflows
- A place to share prompt recipes that actually work

## What Respira is not

- Not "AI magically fixes everything"
- Not a substitute for staging, backups, or basic caution
- Not a place to paste secrets, keys, or client data

## Where to go next

- Quickstart: [Quick Start Guide](https://docs.respira.press/quick-start-guide)
- Ask questions: post in Discussions (Q&A)
- Troubleshooting: post in Discussions (Troubleshooting)
- Bugs: open an Issue using the Bug report form
- Feature requests: post in Discussions (Feature Requests)

Tip: If you include steps to reproduce, you get better answers. Mystery bugs are still mysteries.
