## Welcome

Respira gives you four compatible surfaces:

- **Browser AI** (WebMCP built-in, fast setup)
- **Desktop AI** (standalone MCP server, advanced workflows)
- **WordPress AI stack** (Abilities API + official MCP Adapter)
- **Elementor Angie** (curated browser-auth integration)

All four surfaces share the same duplicate-before-edit safety model where write operations are supported.

## Choose your path

### Path A: Browser AI (fastest)

Best for quick edits, content updates, and team collaboration.

1. Install and activate Respira plugin
2. Open your browser AI client in Chrome 146+
3. Start with a simple request

Continue with: [Browser AI Setup](/browser-ai)

### Path B: Desktop AI (power workflows)

Best for automation, batch operations, and terminal/editor-centric workflows.

1. Install and activate Respira plugin
2. Generate API key in WordPress admin
3. Run MCP setup wizard
4. Connect Cursor/Claude Code/Windsurf

Continue with: [Desktop AI (MCP Server)](/mcp-server)

### Path C: WordPress-native AI stack

Best for teams standardizing on the official WordPress Abilities + MCP Adapter model.

1. Install and activate Respira
2. Confirm the site has the WordPress Abilities API
3. Use the official MCP Adapter default/public server for safe read-only discovery
4. Use the dedicated Respira adapter server when you need the broader registry

Continue with: [WordPress AI Stack Compatibility](/wordpress-ai-stack)

### Path D: Elementor Angie

Best when you already work inside Elementor Angie and want a curated in-browser Respira toolset.

1. Install and activate Respira
2. Enable the Angie integration in plugin settings
3. Open Elementor Angie from logged-in `wp-admin`
4. Start with page discovery or duplicate-first edits

Continue with: [Elementor Angie](/elementor-angie)
