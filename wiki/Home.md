# Respira for WordPress GitHub Community Wiki

**Community-maintained knowledge base for Respira for WordPress.**

This wiki is part of the **Respira for WordPress GitHub Community**. It's written by the community, for the community. If you learned something the hard way, document it here so others don't have to.

## Philosophy

**Give more than you receive.**

If you solved a problem, write it up. If you found a better prompt, share it. If you discovered a gotcha, warn others.

**Socrates' Triple Filter:**
- Is it true? (tested, verified)
- Is it kind? (helps people)
- Is it useful? (actionable)

## Wiki vs Other Channels

**Use the Wiki for:**
- Tested prompt recipes
- Troubleshooting patterns (common issue + proven solution)
- Integration guides (MCP orchestration)
- Builder-specific tips and gotchas
- Real-world case studies

**Use GitHub Discussions for:**
- Asking questions
- Reporting problems
- Requesting features
- Ongoing conversations

**Use Discord for:**
- Quick questions
- Real-time help
- Community chat

**Use Official Docs (respira.press) for:**
- Installation
- Configuration
- API reference
- Official feature documentation

## How to Contribute

1. Find a page that needs improvement (or create a new one)
2. Click "Edit" (top right)
3. Make your changes (follow the templates)
4. Add edit summary: what you changed and why
5. Save

**Quality guidelines:**
- Test before posting (don't guess)
- Include examples (show, don't just tell)
- Redact secrets (API keys, domains, client data)
- Link to official docs when relevant
- Keep it focused (one topic per page)
