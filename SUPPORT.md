# Support

This community is best-effort support for Respira for WordPress.

If you're posting because something is broken, help us help you. The better the report, the faster the fix.

## Where to post what

- How do I...? -> Discussions (Q&A)
- I get an error / something conflicts -> Discussions (Troubleshooting)
- Confirmed bug with steps -> Issues (Bug Report form)
- Feature request -> Discussions (Feature Requests)

## Redaction rules (seriously)

Do NOT post:

- license keys
- API keys or tokens
- passwords
- wp-admin URLs that expose sensitive paths
- private client domains (use example.com or REDACTED)
- full server logs with secrets

If you accidentally post something sensitive, edit or delete the post immediately.

## Response expectations

- Community replies are best effort.
- Maintainer replies are not guaranteed in real time.
- If you need priority support, use your paid support channel (if applicable) or hire a professional.
